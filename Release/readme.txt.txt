=== ZEN L'initiée ===

Attention le répertoire data doit se trouver au même emplacement que le JAR.

** Exécution du jeu
Se placer à l'aide du terminal dans le répertoire contenant le JAR et écrire la commande :

java -jar zenGame.jar

** Choix du mode de jeu
Une fois le jeu lancé vous pouvez choisir entre l'interface graphique ou console.

Si vous souhaitez uniquement jouer en interface graphique il est possible de double cliquez tout simplement sur le JAR.

** Fonctionnement de l'IA
L'ia fonctionne tout simplement en jouant au hasard ainsi il est relativement simple de gagner une partie.
